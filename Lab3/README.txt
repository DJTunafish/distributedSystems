File descriptions
---------------------------------------
sendRequest.sh: Script for sending requests to servers. Takes 2 arguments - 1. number of servers 2. number of requests per server
API-description.txt: Description of the routes used in the server
Report.txt: Contains reports for all tasks


-----------------------------------
Unit test script:
-----------------------------------
Unit tests located in testConsistency.sh
To run tests, start up the lab3.py mininet topology, then run
testConsistency.sh from any client. The script takes one argument - the number of servers in the network.
